# Aplikasi Mobile UTS - Pemrograman Mobile
Repository ini dibuat untuk memenuhi syarat penilaian pada mata kuliah IFB-355 Pemrogaman Mobile

## Deskripsi Aplikasi

Aplikasi mobile yang menampilkan berbagai fitur:
- Splash Screen dengan informasi mahasiswa
- Dashboard dengan Bottom Navigation
- Halaman Biodata
- Halaman Kontak
- Halaman Kalkulator
- Halaman Cuaca
- Halaman Berita

## Fitur yang Sudah Dikembangkan

### 1. Splash Screen
Halaman awal aplikasi yang menampilkan:
- Judul aplikasi
- Foto mahasiswa
- NIM
- Nama mahasiswa
- Durasi: 5 detik
